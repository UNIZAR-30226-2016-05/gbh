Configuración del Sistema

	El sistema será un proyecto de Eclipse Mars lo que nos facilitará un poco la configuración de este.

	Elementos:
		- Archivos de Bootstrap para la interfaz de la página web.
		- Archivos Semantic-UI para la interfaz de la página web.
		- Librería para conectar con la base de datos mysql.
		- Archivos necesarios del servidor tomcat.

	Ubicación:
		- Los archivos de Bootstrap estarán ubicados dentro de la carpeta WebContent del proyecto y dentro una carpeta.
		- Los archivos de Semantic-UI estarán ubicados dentro de la carpeta WebContent del proyecto y dentro una carpeta.
		- La librería de mysql estará dentro de la carpeta lib, que es una subcarpeta de WEB-INF que a su vez es una subcarpeta de la carpeta WebContent del proyecto.
		- Los archivos del necesarios para tomcat estarán almacenados dentro del workspace donde está el proyecto en una carpeta denominada Servers.

	Formato, nombrado y versionado:
		Los elementos tendrán el formato que sea necesario para que puedan funcionar. El nombre será el original y la versión tiene que ser la que funcione para todos los miembros correctamente. 

	Copias de seguridad:
		Cada miembro que modifique algún elemento de configuración lo tendra que subir al repositorio git, después de haberlo probado satisfactoriamente, y lo deberá notificar al resto y estos tendrán que guardar estos cambios en su equipo. Esto nos permitirá tener 5 copias de seguridad de cada elemento.
